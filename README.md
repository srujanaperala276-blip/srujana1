# Groq Chatbot

A high-performance chatbot built with React, Node.js, and the Groq API.

## Quick Start

1. **Backend**:
   - `cd server`
   - Create `.env` and add `GROQ_API_KEY=your_key`
   - `node index.js`

2. **Frontend**:
   - `cd client`
   - `npm install` (if not already done)
   - `npm run dev`

Visit [http://localhost:5173](http://localhost:5173) to start chatting!
